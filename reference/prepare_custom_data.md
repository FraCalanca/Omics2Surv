# Prepare custom data in LinkedOmics-like format

Prepare custom data in LinkedOmics-like format

## Usage

``` r
prepare_custom_data(data, data_type = c("Omics", "Clinical"))
```

## Arguments

- data:

  A data.frame or matrix in LinkedOmics-like format

- data_type:

  Character string specifying the data type. Supported values are
  "Omics" and "Clinical".

## Value

A cleaned data.frame ready for downstream analyses.
